# kafka-simulator

Instructions:

1. git clone https://github.com/arktry123/kafka-simulator.git
2. Go to the project folder (kafka-simulator) and run `docker-compose up` to create Kafka cluster locally
3. Run `mvn clean install`
4. run the `SimulatorApplication` from IntelliJ or run using `java -jar target/producer-0.0.1.jar
